# Mon Portfolio

Bienvenue sur mon portfolio ! Ce projet présente mes compétences, projets et expériences.

## À Propos

Ce portfolio est conçu pour mettre en valeur mes réalisations et compétences dans le domaine du développement web.

## Structure du Projet

- `index.html`: Page principale du portfolio.
- `script.js`: Contient la logique JavaScript.
- `styles.css`: Styles CSS pour le portfolio.
- `favicon.png`: Icône du portfolio.
- `Tirage/`: Dossier contenant des fichiers liés au tirage.
- `LAB2R/`: Dossier pour le laboratoire 2.
- `LAB3R/`: Dossier pour le laboratoire 3.
- `TP1/`: Dossier pour le TP1.
- `LAB1R/`: Dossier pour le laboratoire 1.
- `TP2/`: Dossier pour le TP2.

## Installation

Pour installer et exécuter ce portfolio localement, suivez ces étapes :

1. Clonez ce dépôt sur votre machine locale.
2. Ouvrez `index.html` dans votre navigateur pour voir le portfolio.

## Utilisation

Le portfolio est conçu pour être simple et intuitif. Naviguez à travers les différentes sections pour découvrir mes projets et compétences.
